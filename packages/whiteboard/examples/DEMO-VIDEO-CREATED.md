## 🎨 设计特点

### 1. 分区布局
```
左侧 (300-850px)   → 4个核心特性
右侧 (1050-1650px) → 组件展示
中央 (510-1410px)  → 标题 + CTA
```

### 2. 视觉层次
```
Level 1: 大标题 (72px)
Level 2: 小标题 (64px)
Level 3: 特性标题 (42px)
Level 4: 描述文字 (28px)
```

### 3. 颜色语义
```
绿色 → 开源/免费/成功
蓝色 → 技术/可靠
紫色 → AI/创新
橙色 → 功能/丰富
红色 → 重要/行动
```

### 4. 动画节奏
```
快速: 20-30 frames (0.7-1s)  → 小元素
标准: 60-90 frames (2-3s)    → 主要内容
强调: 90+ frames (3s+)       → 重点信息
```
