## 🎬 使用方法

### 1. 开发预览

```bash
# 安装依赖
cd packages/whiteboard
npm install

# 构建
npm run build

# 查看示例
# (需要配置开发服务器)
```

### 2. 导出视频

```bash
# 使用渲染器导出
# (需要实现渲染器)
seqvio render examples/04-framework-intro.tsx --output intro.mp4
```

## 📝 代码亮点

### 1. 清晰的结构

```tsx
// 场景划分清晰
{/* ===== SCENE 1: Title ===== */}
{/* ===== SCENE 2: Key Features ===== */}
```

### 2. 统一的时间轴

```tsx
// 所有动画按序排列
start={0}      // 0秒
start={120}    // 4秒
start={210}    // 7秒
```

### 3. 颜色系统

```tsx
// 一致的颜色使用
strokeColor="#27ae60"  // 开源 = 绿色
strokeColor="#3498db"  // React = 蓝色
strokeColor="#9b59b6"  // AI = 紫色
```

### 4. 动画细节

```tsx
// 填充延迟效果
fillColor="#3498db"
fillDelay={0.4}

// 手绘效果
roughness={2}

// 描边粗细
strokeWidth={8}  // 勾选标记加粗
```

## 🎨 可以改进的地方

### 如果你想定制：

1. **修改颜色**
   ```tsx
   strokeColor="#你的颜色"
   ```

2. **调整时间**
   ```tsx
   duration={你想要的帧数}
   ```

3. **改变布局**
   ```tsx
   position={{ x: 新位置, y: 新位置 }}
   ```

4. **添加更多内容**
   - 更多特性
   - 更多示例
   - 更多图形

5. **改变风格**
   ```tsx
   texture="whiteboard"  // 或 chalkboard
   roughness={5}         // 更手绘的效果
   ```
