## Getting Help

- Open a GitHub issue for bugs, missing docs, or feature gaps in current behavior.
- Include the exact command, file path, and error output when reporting renderer or audio workflow problems.
- For security-sensitive issues, follow [SECURITY.md](./SECURITY.md) instead of posting full details publicly.
